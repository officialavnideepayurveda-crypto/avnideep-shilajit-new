💳 Razorpay

Accept online payments via Razorpay (UPI, Credit/Debit Card, NetBanking, Wallet)

Enable Razorpay
Allow customers to pay online via Razorpay
Use Test Mode for development, Live Mode for real payments
Find this in Razorpay Dashboard → Settings → API Keys
⚠️ Secrets are stored as Worker Secrets
The following must be set via the Cloudflare Dashboard or CLI:
wrangler secret put RAZORPAY_KEY_SECRET — Your Razorpay Key Secret
wrangler secret put RAZORPAY_WEBHOOK_SECRET — Webhook secret for payment events 

# Deploy secrets to your admin API worker:
cd admin-panel/admin-api-worker
npx wrangler secret put RAZORPAY_KEY_SECRET
# Paste your Razorpay Key Secret when prompted

npx wrangler secret put RAZORPAY_WEBHOOK_SECRET
# Paste your Webhook Secret when prompted

# For the main website webhook:
cd Avnideep-Shilajit
npx wrangler secret put RAZORPAY_WEBHOOK_SECRET
